# 📁 Burlak

> Клиентская часть для загрузки больших файлов на сервер с поддержкой чанков и докачки

---

## 🚀 О проекте

**Burlak** — это современное веб-приложение для загрузки больших файлов на сервер. Проект реализован на **Vue 3** с использованием **TypeScript** и **Vite**.


### Основные возможности

- 📤 **Drag & Drop** — перетаскивание файлов мышью
- 📁 **Выбор файлов** через стандартный диалог
- 📊 **Прогресс загрузки** в реальном времени
- 🎯 **Статусы** загрузки (загрузка, готово, ошибка)
- 🗑 **Управление списком** загруженных файлов
- 📥 **Скачивание результата** (в разработке)

## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <url-репозитория>
cd burlak
2. Установить зависимости
bash
npm install
3. Запустить в режиме разработки
bash
npm run dev
Приложение будет доступно по адресу: http://localhost:3000

4. Собрать для продакшена
bash
npm run build
Собранные файлы появятся в папке dist/.

🧪 Тестирование
Проект пока в стадии разработки. Для тестирования используется симуляция загрузки (без реального бэкенда).

Чтобы протестировать с реальным бэкендом:

Запусти бэкенд на http://localhost:8080

Раскомментируй прокси в vite.config.ts

Перезапусти проект
